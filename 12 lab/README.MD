# Lab 12: Abstract Classes

## Objective
Practice creating and using abstract classes in Python.

## Tasks
1. **Shapes**: Abstract class `Shape` with methods `area()` and `perimeter()`. Implemented `Circle` and `Rectangle`.
2. **Employees**: Abstract class `Employee` with method `calculate_salary()`. Implemented `Manager` and `Developer`.
3. **Library Items**: Abstract class `Item` with method `info()`. Implemented `Book` and `Magazine`.
